◆ OnConfirm1()

void Gui::OnConfirm1 ( )
protected

Handles confirm button press from layout 1.

The controller is queried to read the beginning of the input file. If that succeeds, controller is put in control of the program flow to read the full header information and proceed as it determines to from there. If the read fails, the user is informed that the file was invalid and the interface state is set back to SetLayout1().
