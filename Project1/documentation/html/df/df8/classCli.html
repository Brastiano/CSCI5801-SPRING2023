◆ SetLayout1()

void Cli::SetLayout1 ( )

Set the interface to Layout 1 (date and input file name entry)

The controller is queried to read the beginning of the input file. If that succeeds, controller is put in control of the program flow to read the full header information and proceed as it determines to from there. If the read fails, the user is informed that the file was invalid and the program loops to the beginning of this function to receive fresh input.
